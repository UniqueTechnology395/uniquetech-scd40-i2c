Unique Tech: SCD40 CO2 Sensor Integration (I2C)Αυτό το αποθετήριο περιέχει τον κώδικα, τα ηλεκτρονικά σχέδια και την τεκμηρίωση για τη σύνδεση του αισθητήρα SCD40 (CO2) μέσω πρωτοκόλλου I2C. Το project αποτελεί μέρος του βραβευμένου βιο-ρομποτικού συστήματος της ομάδας Unique Tech (Sparmatseto Robotics).🏆 ACSTAC 2026 Innovation AwardΤο σύστημα αυτό χρησιμοποιήθηκε στο project μας που απέσπασε το Βραβείο Καινοτομίας στο 12ο Μαθητικό Συνέδριο Επιστήμης και Τεχνολογίας του Κολλεγίου Ανατόλια.🚀 ΠεριγραφήΟ αισθητήρας SCD40 χρησιμοποιείται για τη συνεχή παρακολούθηση των επιπέδων Διοξειδίου του Άνθρακα (CO2) σε σχολικές αίθουσες. Τα δεδομένα αυτά τροφοδοτούν ένα μοντέλο Τεχνητής Νοημοσύνης (Explainable AI) για την πρόβλεψη της ευεξίας και της πνευματικής συγκέντρωσης των μαθητών.Χαρακτηριστικά:Πρωτόκολλο: I2C (Pins A4/SDA, A5/SCL στο Arduino Uno).Αισθητήρας: Sensirion SCD40 (Φωτοακουστική τεχνολογία).Πλατφόρμα: Arduino Uno / Uno Q.🛠️ Συνδεσμολογία (Wiring)SCD40 PinArduino PinVCC5V / 3.3VGNDGNDSCLA5SDAA4💻 ΕγκατάστασηΚατεβάστε το αρχείο .ino από το φάκελο /src.Εγκαταστήστε τη βιβλιοθήκη Adafruit SCD4X μέσω του Arduino Library Manager.Ανεβάστε τον κώδικα στο Arduino σας.🔗 ΣύνδεσμοιΕπίσημο Site: uniquetech.carrd.coBlog & Tutorials: [Το URL του blog σας]Εργαστήριο: Sparmatseto Robotics
